# The project

```                     
----------------------------------
| ::::::::::::::::::::::::::::::  |
|                    5            |
|           -----                 |
|           | 1 |             6 : |
|           -----  4      3       |
|                ____   ____      |
|                |  |   |  |  _ 2 |
-----------------|  |---|  |--||---
```
1. RISC-V processor
2. AUX audio
3. USB-type B
4. Another USB-type B
5. 60 GPIO
6. 2 Power pins (+ and -)

It's based on the Raspberry Pi mostly.
